Variables y background

Variables CSS (custom properties)

Una variable es un espacio en memoria en el que guardamos un valor para poder utilizarlo o modificarlo.
Las variables necesitan estar dentro de un selector.
Tienen herencia y cascada.
Existen varibles globales y locales, eso dependerá del selector el que la declaremos.
Las varibles CSS no son lo mismo que las variables SCSS.
Su sintaxis:

selector{
--nombreVariable: valor;
}
propiedad: var(--nombreVariable)

Background

Es una propiedad que nos permite darle fondo a las cajas.
La propiedad background es un shorthand, la lista de propiedades que incluye background son:

.background-color: Nos permite establecer el color de fondo.
.background-image: Nos permite establecer una imagen de fondo.
.background-repeat: Nos permite establecer si un fondo se repite o no.
.background-position: Nos permite establecer la posición del background.
.background-size: Nos permite establecer el tamaño del background.
.background-origin: Nos permite establecer desde donde se empezará a dibujar el fondo.
.background-clip: Nos permite establecer en qué parte de la caja se dibujará el fondo.
.background-attachment: Nos permite establecer si el backgroun es relativo a la caja o al viewport.
